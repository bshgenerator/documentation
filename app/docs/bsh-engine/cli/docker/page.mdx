# Docker Commands

The `docker` command enables you to setup Docker Compose and environment files for BSH Engine.

## `docker`

### `setup`

Setup Docker Compose and environment files for BSH Engine. this will create `docker-compose.yml` and `.env` files like in [installation](/docs/bsh-engine/installation#docker-compose-file)

<Tabs items={['npm', 'npx']}>
    <Tabs.Tab title="npm">
    ```bash
    bshg docker setup [options]
    ```
    </Tabs.Tab>
    <Tabs.Tab title="npx">
    ```bash
    npx @bshsolutions/cli docker setup [options]
    ```
    </Tabs.Tab>
</Tabs>

| Option | Description | Required | Default |
|--------|-------------|----------|---------|
| `-f, --force` | Overwrite existing files | no | false |
| `-o, --output-dir <dir>` | Output directory (default: current directory) | no | `.` |
| `-h, --help` | Display help for command | no | false |
