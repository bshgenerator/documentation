# Plugins Commands

The `plugins` command enables to manage [BSH Engine plugins](/docs/bsh-engine/packaging/plugins) from the command line.

## `install`

Install a plugin from a directory.

<Tabs items={['npm', 'npx']}>
    <Tabs.Tab title="npm">
    ```bash
    bshg plugins install <plugin-dir> [options]
    ```
    </Tabs.Tab>
    <Tabs.Tab title="npx">
    ```bash
    npx @bshsolutions/cli plugins install <plugin-dir> [options]
    ```
    </Tabs.Tab>
</Tabs>

<Callout type="info">Alias: `bshg plugins i`</Callout>

| Argument | Description | Required | Default |
|----------|-------------|----------|---------|
| `<plugin-dir>` | Path to the plugin directory | yes | ~ |

| Option | Description | Required | Default |
|--------|-------------|----------|---------|
| `-h, --host <host>` | BSH Engine host URL | yes | ~ |
| `-k, --api-key <key>` | BSH Engine API key | yes | ~ |
| `--no-verbose` | Disable verbose output (default: verbose enabled) | no | false |

**Usage Example:**

<Tabs items={['npm', 'npx']}>
    <Tabs.Tab title="npm">
    ```bash
    bshg plugins install ./path/to/plugin \
      --host https://instance-host.com \
      --api-key <api-key>
    ```
    </Tabs.Tab>
    <Tabs.Tab title="npx">
    ```bash
    npx @bshsolutions/cli plugins install ./path/to/plugin \
      --host https://instance-host.com \
      --api-key <api-key>
    ```
    </Tabs.Tab>
</Tabs>
