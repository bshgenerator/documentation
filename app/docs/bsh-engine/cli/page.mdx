# CLI Documentation

## Overview

`bshg` is the BSH Engine Manager CLI tool for managing BSH Engine plugins. It provides commands to install and manage plugins for the BSH Engine platform.

## Installation

Install the CLI globally:

<Tabs items={['npm', 'npx']}>
    <Tabs.Tab title="npm">
```bash
npm install -g @bshsolutions/cli
```
    </Tabs.Tab>
    <Tabs.Tab title="npx">
```bash
npx @bshsolutions/cli [options] [command]
```
    </Tabs.Tab>
</Tabs>

## Usage

<Tabs items={['npm', 'npx']}>
    <Tabs.Tab title="npm">
```bash
bshg [options] [command]
```
    </Tabs.Tab>
    <Tabs.Tab title="npx">
```bash
npx @bshsolutions/cli [options] [command]
```
    </Tabs.Tab>
</Tabs>

<Callout type="">
  - `-V, --version` - Display version number
  - `-h, --help` - Display help for command
</Callout>

### Commands

| Command | Description |
|---------|-------------|
| [`bshg plugins install`](/docs/bsh-engine/cli/plugins#install) | Install a plugin to your BSH Engine instance. |
| [`bshg plugins validate`](/docs/bsh-engine/cli/plugins#validate) | Validate a plugin's structure and correctness. |
| [`bshg plugins generate`](/docs/bsh-engine/cli/plugins#generate) | Generate a new plugin. |
