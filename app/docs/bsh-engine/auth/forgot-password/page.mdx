# Forgot Password

## Request Password Reset Code

if you forgot your password, you can request a password reset code via email.

![Forgot Password Page](/images/bsh-egnine/auth/forgot-password.png)

<Callout type='info'>
    For detailed request and response formats, refer to the [Authentication API Documentation](/docs/bsh-engine/api/authentication/forgot-password).
</Callout>

## Reset Your Password

after you request a password reset code, you will receive an email with a code to reset your password.

![Reset Password Page](/images/bsh-egnine/auth/reset-password.png)

<Callout type='warning'>
    the reset password code can be used only one time.
</Callout>

<Callout type='info'>
    For detailed request and response formats, refer to the [Authentication API Documentation](/docs/bsh-engine/api/authentication/reset-password).
</Callout>
