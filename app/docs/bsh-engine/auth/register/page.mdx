# Register new user

To register a new user, you can use the register form in the admin panel.

![Register Page](/images/bsh-egnine/auth/register.png)

<Callout type='info'>
    For detailed request and response formats, refer to the [Register API Documentation](/docs/bsh-engine/api/authentication/register).
</Callout>

<Callout type='warning'>
    To enable the register feature, you must set the **Enable Register** setting to true in the [Engine Settings](/docs/bsh-engine/settings).
</Callout>
