# Users API

The Users API provides comprehensive user management operations including profile management, authentication, and file uploads.


<Callout type="info">
  This API is linked with the `BshUsers` entity.
</Callout>


## Base Endpoint

```bash
/api/user
```

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| [POST](/docs/bsh-engine/api/users/init-user) | `/api/user/init` | Initialize a new user account |
| [GET](/docs/bsh-engine/api/users/get-current-user) | `/api/user/me` | Get current user profile |
| [PUT](/docs/bsh-engine/api/users/update-profile) | `/api/user/profile` | Update user profile |
| [PUT](/docs/bsh-engine/api/users/update-password) | `/api/user/password` | Update user password |
| [POST](/docs/bsh-engine/api/users/update-picture) | `/api/user/picture` | Upload profile picture |

## Security

All Users API endpoints are secured by the [authentication](/docs/bsh-engine/api/authentication) system. User management operations require proper authentication tokens and users can only access and modify their own profile data. Unauthorized requests will be denied accordingly.
