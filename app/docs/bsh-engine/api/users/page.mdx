# Users API

The Users API provides comprehensive user management operations including profile management, authentication, and file uploads.


<Callout type="info">
  This API is linked with the `BshUsers` entity.
</Callout>


## Base Endpoint

```bash
/api/user
```

## Endpoints

| Method | Endpoint | Description | |
|--------|----------|-------------|-|
| [POST](/docs/bsh-engine/api/users/init-user) | `/api/users/init` | Initialize a new user account | |
| [GET](/docs/bsh-engine/api/users/get-current-user) | `/api/users/me` | Get current user profile | |
| [PUT](/docs/bsh-engine/api/users/update-profile) | `/api/users/profile` | Update user profile | |
| [PUT](/docs/bsh-engine/api/users/update-password) | `/api/users/password` | Update user password | |
| [POST](/docs/bsh-engine/api/users/update-picture) | `/api/users/picture` | Upload profile picture | |
| [GET](/docs/bsh-engine/api/entities/get-entity-by-id) | `/api/users/{id}` | Get a user by ID | Similar to Entities API |
| [GET](/docs/bsh-engine/api/entities/search-entities-by-query) | `/api/users?query` | List and filter by URI parameters | Similar to Entities API |
| [POST](/docs/bsh-engine/api/entities/search-entities) | `/api/users/search` | Search users by POST | Similar to Entities API |
| [DELETE](/docs/bsh-engine/api/entities/delete-entity) | `/api/users/{id}` | Delete a user | Similar to Entities API |
| [GET](/docs/bsh-engine/api/entities/count-entities) | `/api/users/count` | Count users | Similar to Entities API |
| [POST](/docs/bsh-engine/api/entities/count-filtered-entities) | `/api/users/count` | Count filtered users | Similar to Entities API |

## Security

All Users API endpoints are secured by the [authentication](/docs/bsh-engine/api/authentication) system. User management operations require proper authentication tokens and users can only access and modify their own profile data. Unauthorized requests will be denied accordingly.
