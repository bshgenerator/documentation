# Files API

The Files API provides comprehensive file management operations including image uploads, metadata management, and file organization with cloud storage integration.

<Callout type="info">
  This API is linked with the `BshFiles` entity.
</Callout>

## Base Endpoint

```bash
/api/images
```

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| [POST](/docs/bsh-engine/api/files/upload-image) | `/api/images/upload` | Upload an image file to cloud storage |

<Callout type="info">
  Additional endpoints for file management are planned and will be introduced in future releases. Stay tuned for updates as development continues!
</Callout>
