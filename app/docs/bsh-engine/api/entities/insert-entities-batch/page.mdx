# Batch Insert Entities

Insert multiple records in the entity in a single request, this endpoint will return the inserted records with the [persistenceId](/docs/bsh-engine/api/persistence-id).<br/>

```bash
POST /api/entities/:entityName/batch
```

<Callout type='important'>
  If the table linked to the entity is not exists, the endpoint will create the table and insert the records.<br/>
  It will use the defined primary keys in [Entity Form](/docs/bsh-engine/modeling/entities#primary-keys) and will create new table with the specified primary keys.
</Callout>

## Request Body

```json
[
    { "key": "value" },
    { "key": "value" },
]
```

## Response

```json
{
  "data": [
    { "key": "value", "persistenceId": "..." },
    { "key": "value", "persistenceId": "..." },
  ],
  "status": "OK",
  "code": 200,
  "timestamp": 1234567890,
  "meta": {},
}
```

<Callout type='default'>
  Read the [Persistence ID](/docs/bsh-engine/api/persistence-id) documentation for more information about the `persistenceId`.
</Callout>
