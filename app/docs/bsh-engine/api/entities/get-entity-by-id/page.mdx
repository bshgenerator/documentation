# Get Entity by ID

Get an entity by ID.

```bash
GET /api/entities/:entityName/:entityId
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| entityName | string | The name of the entity to get |
| entityId | string | The ID of the entity to get |

## Response

```json
{
  "data": [
    {
        "...": "...",
        "persistenceId": "123",
    }
  ],
  "status": "OK",
  "code": 200,
  "timestamp": 1234567890,
  "meta": {},
}
```

<Callout type='default'>
  Read the [Persistence ID](/docs/bsh-engine/api/persistence-id) documentation for more information about the `persistenceId`.
</Callout>
