# Settings API

The Settings API provides endpoints for managing application configuration settings. You can retrieve current settings and update them programmatically.

<Callout type="info">
  For dashboard instructions and general information about Settings, see the [Settings documentation](/docs/bsh-engine/settings).
</Callout>

## Base Endpoint

```bash
/api/settings
```

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| [GET](/docs/bsh-engine/api/settings/get-configs) | `/api/settings` | Get current configuration settings |
| [PUT](/docs/bsh-engine/api/settings/update-configs) | `/api/settings` | Update configuration settings |
