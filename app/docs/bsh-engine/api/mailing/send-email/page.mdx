# Send Email

Send an email using either plain text, HTML content, or email templates.

<Callout type="error">
  First you need to configure the [email configurations](/docs/bsh-engine/mailing/configurations) before using this endpoint.
</Callout>

## Endpoint

```bash
POST /api/mailing/send
```

## Request Body

<Tabs items={['Plain Text', 'HTML', 'Template', 'Bulk Email', 'Attachments']}>
<Tabs.Tab title="Plain Text">
  Simple plain text email.
  ```json
  {
    "to": "recipient@example.com",
    "subject": "Test Email",
    "body": "This is a test email body",
    "html": false
  }
  ```
</Tabs.Tab>
<Tabs.Tab title="HTML">
  Send an email with HTML content.
  ```json
  {
    "to": "recipient@example.com",
    "subject": "HTML Test Email",
    "body": "<h1>Hello World</h1><p>This is an <strong>HTML</strong> email.</p>",
    "html": true
  }
  ```
</Tabs.Tab>
<Tabs.Tab title="Template">
  Use [email templates](/docs/bsh-engine/mailing/email-templates) to send an email.
  ```json
  {
    "to": "recipient@example.com",
    "template": "welcome-email",
    "data": {
      "userName": "John Doe",
      "loginUrl": "https://app.example.com/login",
      "supportEmail": "support@example.com"
    }
  }
  ```
</Tabs.Tab>
<Tabs.Tab title="Attachments">
  <Callout type="info">
    This is a planned feature and will be implemented in the future releases.
  </Callout>
  ```json
  {
    "to": "recipient@example.com",
    "subject": "Document Attached",
    "body": "Please find the attached document.",
    "attachments": [
      {
        "fileId": "file-123",
        "fileName": "document.pdf",
        "fileType": "application/pdf",
        "fileSize": 1024
      }
    ]
  }
  ```
</Tabs.Tab>
<Tabs.Tab title="Bulk Email">
  <Callout type="info">
    This is a planned feature and will be implemented in the future releases.
  </Callout>
  ```json
  {
    "to": ["user1@example.com", "user2@example.com", "user3@example.com"],
    "template": "newsletter",
    "data": {
      "userName": "User",
      "newsletterContent": "Latest updates...",
      "unsubscribeUrl": "https://app.example.com/unsubscribe"
    }
  }
  ```
</Tabs.Tab>
</Tabs>

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `to` | string/array | Yes | Recipient email address(es) |
| `subject` | string | No | Email subject line |
| `body` | string | No* | Email body content |
| `html` | boolean | No | Whether body contains HTML (default: false) |
| `template` | string | No* | Template name for template-based emails |
| `data` | object | No | Template variables for dynamic content |

*Either `body` OR `template` must be provided.
