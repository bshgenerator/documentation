# API Keys API

The API Keys API provides endpoints for creating, managing, and revoking API keys programmatically.

<Callout type='info'>
    For dashboard instructions and general information about API Keys, see the [API Keys documentation](/docs/bsh-engine/security/api-keys).
</Callout>

## Base Endpoint

```bash
/api/api-keys
```

## Endpoints

| Method | Endpoint | Description | |
|--------|----------|-------------|-|
| [POST](/docs/bsh-engine/api/api-keys/generate) | `/api/api-keys` | Generate a new API key | |
| [DELETE](/docs/bsh-engine/api/api-keys/revoke) | `/api/api-keys/{id}/revoke` | Revoke an API key | |
| [GET](/docs/bsh-engine/api/entities/get-entity-by-id) | `/api/api-keys/{id}` | Get an API key by ID | Similar to Entities API |
| [GET](/docs/bsh-engine/api/entities/search-entities-by-query) | `/api/api-keys?query` | List and filter by URI parameters | Similar to Entities API |
| [POST](/docs/bsh-engine/api/entities/search-entities) | `/api/api-keys/search` | Search API keys by POST | Similar to Entities API |
| [DELETE](/docs/bsh-engine/api/entities/delete-entity) | `/api/api-keys/{id}` | Delete an API key | Similar to Entities API |
| [GET](/docs/bsh-engine/api/entities/count-entities) | `/api/api-keys/count` | Count API keys | Similar to Entities API |
| [POST](/docs/bsh-engine/api/entities/count-filtered-entities) | `/api/api-keys/count` | Count filtered API keys | Similar to Entities API |
