# BSH Engine SDK - TypeScript

The **BSH Engine SDK for TypeScript** is the official TypeScript/JavaScript library for interacting with BSH Engine API.
It provides a type-safe, developer-friendly interface to all BSH Engine features, making it easy to integrate BSH Engine capabilities into your Node.js, browser, or TypeScript applications.

## Features

- **Type-Safe**: Full TypeScript support with comprehensive type definitions
- **Complete API Coverage**: Access to all BSH Engine endpoints and features
- **Authentication Handling**: Built-in support for API key and token-based authentication
- **Error Handling**: Robust error handling with detailed error messages
- **Request/Response Interceptors**: Customize requests and responses
- **Promise-Based**: Modern async/await support
