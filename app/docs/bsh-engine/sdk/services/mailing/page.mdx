# Mailing Service

The Mailing Service provides methods for sending emails using plain text, HTML content, or email templates.

## Methods

### send

Sends an email using either plain text, HTML content, or email templates.

<Tabs items={['Plain Text', 'HTML', 'Template']}>
<Tabs.Tab title="Plain Text">
  Simple plain text email.
  ```typescript
  await bshEngine.mailing.send({
    payload: {
      to: 'recipient@example.com',
      subject: 'Test Email',
      body: 'This is a test email body',
      html: false
    }
  });
  ```
</Tabs.Tab>
<Tabs.Tab title="HTML">
  Send an email with HTML content.
  ```typescript
  await bshEngine.mailing.send({
    payload: {
      to: 'recipient@example.com',
      subject: 'HTML Test Email',
      body: '<h1>Hello World</h1><p>This is an <strong>HTML</strong> email.</p>',
      html: true
    }
  });
  ```
</Tabs.Tab>
<Tabs.Tab title="Template">
  Use [email templates](/docs/bsh-engine/mailing/email-templates) to send an email.
  ```typescript
  await bshEngine.mailing.send({
    payload: {
      to: 'recipient@example.com',
      template: 'welcome-email',
      data: {
        userName: 'John Doe',
        loginUrl: 'https://app.example.com/login',
        supportEmail: 'support@example.com'
      }
    }
  });
  ```
</Tabs.Tab>
</Tabs>

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `to` | string | yes | Recipient email address |
| `subject` | string | no | Email subject line |
| `body` | string | no* | Email body content |
| `html` | boolean | no | Whether body contains HTML (default: false) |
| `template` | string | no* | Template name for template-based emails. See [Email Templates](/docs/bsh-engine/mailing/email-templates) |
| `data` | object | no | Template variables for dynamic content |

*Either `body` OR `template` must be provided.
