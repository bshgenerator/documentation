# Changelog

Track all notable changes to BSH Engine.

## [0.0.2]
**Released on: 2025-12-01**

**Features:**
 - Introduce the API Keys system ([API](/docs/bsh-engine/api/api-keys) / [Form](/docs/bsh-engine/security/api-keys))
 - Add User Update ([API](/docs/bsh-engine/api/users/update-profile) / [Form](/docs/bsh-engine/security/users#update-user))
 - Implement Register ([API](/docs/bsh-engine/api/authentication/register) / [Form](/docs/bsh-engine/auth/register))
 - Introduce Engine Settings ([API](/docs/bsh-engine/api/settings) / [Form](/docs/bsh-engine/settings))
 - Powered GET requests with search by [query parameters](/docs/bsh-engine/api/entities/search-entities-by-query)
 - Implement TS SDK for quick access to Engine APIs ([SDK](/docs/bsh-engine/sdk))
 - Update Pagination to be 1-based.

**Improvements:**
 - use new TS library in the dashboard to consume the Engine APIs
 - Improvements in the Dashboard UI/UX
 - Internal caching improvements
 - Fix bugs and improve performance

## [0.0.1]

**Features:**

 - Security: Users, Roles, Policies
 - Files: Configurations, Load File
 - Mailing: Configurations, Email Templates, Send Email

 - Api Documentations
